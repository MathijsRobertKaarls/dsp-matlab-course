3.1 — SNR Improvement

Module 3 · Signal Quality · ~20 min

Introduction

Every real-world measurement is contaminated by noise — thermal fluctuations, electromagnetic interference, sensor imperfections. The Signal-to-Noise Ratio (SNR) quantifies how much of what you're measuring is actual signal versus unwanted noise. A higher SNR means a cleaner, more reliable measurement.

DSP offers several strategies to improve SNR: filtering, averaging, and more. Understanding SNR in decibels is essential for quantifying these improvements.

Core Concept

What is SNR?

SNR is the ratio of signal power to noise power:

$$\text{SNR} = \frac{P_{signal}}{P_{noise}}$$

In practice we use the decibel (dB) scale because SNR values span many orders of magnitude:

$$\text{SNR}_{dB} = 10 \log_{10}\!\left(\frac{P_{signal}}{P_{noise}}\right) = 20 \log_{10}\!\left(\frac{A_{signal}}{A_{noise}}\right)$$

Common reference points:

  • 0 dB — signal and noise are equal power (barely usable)
  • 10 dB — signal is 10× stronger than noise (audible)
  • 40 dB — signal is 10,000× stronger (high quality)
  • -∞ dB — pure noise, no signal

SNR Improvement Strategies

  • Filtering — reject noise outside the signal's frequency band (covered in Module 5)
  • Averaging — average $M$ repeated measurements; SNR improves by $\sqrt{M}$
  • Coherent detection — lock to a known reference phase (lock-in amplifier)
  • Oversampling — sample faster than necessary, then decimate; gains $\approx 3$ dB per doubling

Averaging: SNR ∝ √M

If you repeat a measurement $M$ times and average the results, random noise partially cancels while the coherent signal adds up:

$$\text{SNR}_{avg} = \sqrt{M} \cdot \text{SNR}_{single}$$ $$\text{SNR}_{dB,avg} = \text{SNR}_{dB,single} + 10\log_{10}(M) \;\text{dB}$$
Example: Averaging 100 ECG sweeps improves SNR by $10\log_{10}(100) = 20$ dB — a 100× power improvement.

Mathematical Foundation

Signal model: $x[n] = s[n] + w[n]$, where $s[n]$ is the deterministic signal and $w[n]$ is zero-mean noise with variance $\sigma^2$.

After averaging $M$ trials:

$$\bar{x}[n] = \frac{1}{M}\sum_{m=1}^{M} x_m[n]$$

The signal component (coherent) adds constructively: mean stays $s[n]$. The noise variance reduces: $\text{Var}(\bar{w}) = \sigma^2/M$. So noise amplitude scales as $1/\sqrt{M}$, giving SNR ∝ $\sqrt{M}$.

MATLAB Exercise

%% 3.1 SNR Improvement by Averaging

fs = 1000; T = 0.5;
t  = 0:1/fs:T-1/fs;
f0 = 50;  A = 1;

% True signal
s = A * sin(2*pi*f0*t);

% Simulate M noisy measurements
M = 64;
sigma = 2;   % noise std dev → SNR_single ≈ 20·log10(A/sigma) ≈ -6 dB
rng(42);     % reproducible

measurements = s + sigma * randn(M, length(t));

% Compute running average and SNR
snr_db = zeros(1, M);
avg = zeros(size(t));
for m = 1:M
  avg = avg + measurements(m,:);
  x_avg = avg / m;
  noise_est = x_avg - s;
  snr_db(m) = 10*log10(sum(s.^2) / sum(noise_est.^2));
end

% Plot
figure;
subplot(3,1,1); plot(t, measurements(1,:)); title('Single noisy measurement'); ylabel('Amplitude'); ylim([-6 6]);
subplot(3,1,2); plot(t, avg/M); hold on; plot(t, s, '--'); title(sprintf('Average of %d measurements', M));
legend('Average','True signal'); ylim([-2 2]);
subplot(3,1,3); plot(1:M, snr_db, 'b-o', 'MarkerSize', 3); hold on;
plot(1:M, 10*log10(1:M) + snr_db(1), 'r--');
xlabel('Number of averages M'); ylabel('SNR (dB)');
title('SNR vs M — blue: measured, red: theoretical √M improvement');
legend('Measured SNR','Theoretical');

% MATLAB built-in SNR function (Signal Processing Toolbox)
fprintf('SNR (1 measurement):  %.1f dB\n', snr(s, measurements(1,:)-s));
fprintf('SNR (%d measurements): %.1f dB\n', M, snr(s, avg/M - s));
💡 Hint

The SNR plot should follow the red dashed theoretical line closely. Each time you quadruple M, SNR improves by ~6 dB. At M=64, you should gain about 18 dB over a single measurement.

Key Takeaways

  • SNR measures signal quality: $\text{SNR}_{dB} = 10\log_{10}(P_s/P_n)$.
  • Averaging $M$ measurements improves SNR by $\sqrt{M}$ (or $10\log_{10}M$ dB).
  • Filtering, oversampling, and coherent detection are complementary SNR strategies.
  • Each extra 6 dB of SNR requires 4× more averages — expensive at high SNR!
