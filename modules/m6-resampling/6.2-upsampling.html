6.2 — Upsampling

Module 6 · Resampling · ~15 min

Introduction

Upsampling increases the number of samples — you go from a lower to a higher sampling rate. The challenge: the new samples don't exist yet. Naively inserting zeros between samples creates an ugly, distorted-looking signal. Upsampling done correctly requires inserting zeros and then smoothing with an interpolation (lowpass) filter.

Core Concept

The Upsampling Process

Upsampling by integer factor $L$:

  1. Zero-insert: insert $L-1$ zeros between each original sample.
  2. Lowpass filter (interpolation filter): cutoff at the original Nyquist $f_s/2$ (equivalently $\pi/L$ in normalized digital frequency on the new rate $L f_s$), gain $L$ to restore amplitude.
$$x_L[n] = \begin{cases} x[n/L] & n = 0, L, 2L, \ldots \\ 0 & \text{otherwise} \end{cases}$$

Spectral Effect of Zero-Insertion

Inserting zeros creates spectral images — copies of the original spectrum repeated at multiples of the original Nyquist frequency. The lowpass interpolation filter removes these images, leaving only the baseband signal at the higher rate.

Mathematical Foundation

The DTFT of the zero-inserted signal $x_L[n]$:

$$X_L(e^{j\hat\omega}) = X(e^{j\hat\omega L})$$

This is the original spectrum compressed by factor $L$ in digital frequency — equivalently, spectral images appear at $\hat\omega = 2\pi k/L$ for $k = 1, 2, \ldots, L-1$. The interpolation filter $H(e^{j\hat\omega})$ removes everything above $\pi/L$:

$$H_{interp}(\hat\omega) = \begin{cases} L & |\hat\omega| \leq \pi/L \\ 0 & \text{otherwise} \end{cases}$$

The gain $L$ compensates for the energy reduction caused by inserting zeros.

Conceptual Questions

Q1. After zero-insertion (upsampling by L=4 without filtering), the spectrum contains:

  • The original spectrum stretched by a factor of 4
  • The original baseband spectrum plus 3 spectral images at higher frequencies
  • A completely flat (white) spectrum
  • The original spectrum compressed by 4

Q2. The interpolation lowpass filter after zero-insertion must have a gain of $L$ (not 1) because:

  • It needs to compensate for the filter's passband ripple
  • Inserting $L-1$ zeros dilutes the signal energy; gain $L$ restores the correct amplitude
  • The cutoff frequency changes by L
  • It prevents the signal from being aliased

Q3. For upsampling by $L=3$ at original rate $f_s$, the interpolation filter cutoff is:

  • $f_s / 2$ (original Nyquist)
  • $f_s / 6$ (= original Nyquist divided by $L$)
  • $3 f_s / 2$
  • $f_s$

MATLAB Exercise

⚠ Requires Signal Processing Toolbox — upsample, interp, resample, and fir1. 
%% 6.2 Upsampling: Zero Insertion + Interpolation Filter

fs = 1000;
L  = 4;      % Upsample factor
T  = 0.5;
t  = (0:1/fs:T-1/fs);

f0 = 100;
x  = sin(2*pi*f0*t) + 0.5*sin(2*pi*3*f0*t);

%% Step 1: Zero insertion (no filtering)
x_up_raw = upsample(x, L);   % inserts L-1 zeros between samples
fs_up    = fs * L;
t_up     = (0:length(x_up_raw)-1) / fs_up;

%% Step 2: Apply interpolation filter
% Lowpass at fs/(2L), gain = L
b_interp = L * fir1(64, 1/L);   % normalized cutoff = 1/L
x_up_filtered = filter(b_interp, 1, x_up_raw);

%% Plot time domain
figure('Name','Upsampling');
subplot(3,1,1);
stem(t(1:60), x(1:60), 'filled', 'MarkerSize',4);
title(sprintf('Original signal: fs=%d Hz', fs)); xlabel('Time (s)');

subplot(3,1,2);
stem(t_up(1:240), x_up_raw(1:240), 'filled', 'MarkerSize',2);
title(sprintf('After zero insertion (L=%d): fs=%d Hz — zeros visible', L, fs_up));
xlabel('Time (s)');

subplot(3,1,3);
plot(t_up(1:240), x_up_filtered(33:33+239));  % remove filter delay
title(sprintf('After interpolation filter: smooth at %d Hz', fs_up));
xlabel('Time (s)');

%% Spectral comparison
N = length(x);  Nu = length(x_up_raw);
figure('Name','Upsampling Spectra');
ko = 1:floor(N/2);   f_orig = (ko-1)*fs/N;       % slice magnitudes to match axes
ku = 1:floor(Nu/2);  f_up   = (ku-1)*fs_up/Nu;

Xo  = abs(fft(x))*2/N;
Xur = abs(fft(x_up_raw))*2/Nu;
Xuf = abs(fft(x_up_filtered))*2/Nu;   % filter already supplies gain L (b_interp = L*fir1)

subplot(3,1,1);
plot(f_orig, Xo(ko));
title('Original spectrum'); xlabel('Hz'); xlim([0 600]);

subplot(3,1,2);
plot(f_up, Xur(ku));
title('After zero insertion: images visible at multiples of fs/2'); xlabel('Hz'); xlim([0 2000]);
for k = 1:L; xline(fs/2*k, 'r--'); end

subplot(3,1,3);
plot(f_up, Xuf(ku));
title('After interpolation filter: images removed'); xlabel('Hz'); xlim([0 2000]);
xline(fs/2, 'g--', 'original Nyquist');
💡 Hint

The middle spectrum plot will show the original 100 Hz and 300 Hz peaks, PLUS their images at 900/700 Hz, 1900/1700 Hz etc. (images at fs-f and 2fs-f etc.). The bottom plot shows them removed. Note: upsample(x, L) in MATLAB inserts zeros; this is equivalent to zero-stuffing.

Expected outputs

  • The original signal (fs = 1000 Hz) has peaks at $f_0 = 100$ Hz and $3f_0 = 300$ Hz; both are below the original Nyquist of $500$ Hz.
  • upsample(x, 4) inserts $L-1 = 3$ zeros between each sample, raising the rate to fs_up = 4000 Hz and quadrupling the length. The zero-stuffed time plot is visibly discontinuous (samples interleaved with zeros).
  • The zero-stuffed spectrum keeps the $100$ Hz and $300$ Hz baseband peaks and adds $L-1 = 3$ spectral images. Within $[0, 2000]$ Hz the images of the $100$ Hz tone appear at $900$, $1100$, $1900$ Hz (at $f_s\!-\!f_0$, $f_s\!+\!f_0$, $2f_s\!-\!f_0$…) and the $300$ Hz tone at $700$, $1300$, $1700$ Hz — copies spaced about the multiples of $f_s/2 = 500$ Hz marked by the red dashed lines.
  • After the interpolation filter b_interp = L*fir1(64, 1/L) (cutoff at the original Nyquist $f_s/2 = 500$ Hz, i.e. normalized $1/L$, gain $L$) only the $100$ Hz and $300$ Hz baseband peaks survive; all images above $500$ Hz are removed and the time-domain signal becomes a smooth interpolated waveform at $4000$ Hz.

Key Takeaways

  • Upsampling by $L$: insert $L-1$ zeros, then apply lowpass interpolation filter (gain $L$, cutoff $\pi/L$).
  • Zero-insertion creates spectral images — copies of the original spectrum that must be filtered out.
  • In MATLAB: upsample(x, L) for zero-insertion; follow with filter() for interpolation.
  • Or use resample(x, L, 1) which does both steps in one call.
