6.4 — Zero Stuffing

Module 6 · Resampling · ~15 min

Introduction

Zero stuffing (also called zero insertion or zero-padding in the time domain) is the step of inserting zeros between samples when upsampling. It's the first step in the upsampling process we saw in Topic 6.2, but it's important enough to understand in isolation — especially to see exactly what it does to the spectrum and why an interpolation filter is essential afterward.

Core Concept

What Zero Stuffing Does

For an upsampling factor $L$, zero stuffing creates:

$$x_L[n] = \begin{cases} x[n/L] & n = 0, L, 2L, \ldots \\ 0 & \text{otherwise} \end{cases}$$

Visually: every original sample is followed by $L-1$ zeros. The result is a signal at $L$ times the original sample rate — but it's not yet a properly interpolated signal. It looks like the original signal with gaps.

Spectral Consequence: Images

In the frequency domain, zero stuffing compresses the spectrum — but periodically repeats it. The $L \times$ upsampled signal has the same baseband content, plus $L-1$ spectral copies (images) spaced at multiples of the original Nyquist:

$$X_L(e^{j\hat\omega}) = X(e^{jL\hat\omega})$$

This periodicity means the original spectrum repeats $L$ times across the new $[0, \pi]$ range. An interpolation lowpass filter removes all but the baseband copy.

Key distinction from zero-padding (Topic 3.2 / 6.5):
Zero stuffing inserts zeros between samples in the time domain → used for upsampling.
Zero padding appends zeros to the end of a signal → used for FFT interpolation in frequency domain.

Mathematical Foundation

Zero stuffing is equivalent to multiplying the original signal by a comb function (impulse train with period $L$). In the frequency domain, multiplication by an impulse train corresponds to convolution with another impulse train, creating periodic repetitions of the spectrum:

$$X_L(e^{j\hat\omega}) = X(e^{j\hat\omega L})$$

This is the original spectrum evaluated at $L$ times the digital frequency — meaning it's compressed by $L$ and thus repeated $L$ times across $[0, 2\pi]$.

Conceptual Questions

Q1. How is zero stuffing different from zero padding?

  • They are the same operation described differently
  • Zero stuffing inserts zeros between samples (upsampling); zero padding appends zeros at the end (DFT interpolation)
  • Zero stuffing is applied in the frequency domain
  • Zero padding changes the sampling rate; zero stuffing does not

Q2. After zero stuffing by $L=3$, the spectrum contains:

  • Only the baseband signal (no images)
  • The original baseband plus 2 additional spectral images
  • The original spectrum stretched by 3×
  • An aliased signal

Q3. Why is zero stuffing alone insufficient for correct upsampling?

  • It changes the signal amplitude
  • The inserted zeros create spectral images and discontinuities; an interpolation filter is needed to remove images and fill in correct sample values
  • It aliases the signal
  • It only works for integer upsampling factors

MATLAB Exercise

%% 6.4 Zero Stuffing vs Proper Upsampling

fs = 500;
L  = 4;
T  = 0.1;
t  = (0:1/fs:T-1/fs);
f0 = 80;
x  = sin(2*pi*f0*t);

%% Method A: upsample() — zero stuffing only (MATLAB)
x_stuffed = upsample(x, L);   % inserts L-1 zeros between each sample
fs_new    = fs * L;
t_new     = (0:length(x_stuffed)-1)/fs_new;

%% Method B: zero stuffing + interpolation filter
b_interp   = L * fir1(64, 1/L);
x_interp   = filter(b_interp, 1, x_stuffed);

%% Method C: resample() — polyphase (most accurate)
x_resamp = resample(x, L, 1);

%% Plots
t_ref = (0:1/(fs*100):T);
x_ref = sin(2*pi*f0*t_ref);   % very fine "true" reference

figure('Name','Zero Stuffing vs Proper Upsampling');
subplot(4,1,1);
stem(t, x, 'filled', 'MarkerSize', 5); hold on;
plot(t_ref, x_ref, 'k:', 'LineWidth', 0.5);
title(sprintf('Original: fs=%d Hz', fs)); xlim([0 0.06]);

subplot(4,1,2);
stem(t_new, x_stuffed, 'filled', 'MarkerSize', 2);
title(sprintf('After zero stuffing (L=%d, fs=%d Hz) — discontinuous!', L, fs_new)); xlim([0 0.06]);

subplot(4,1,3);
plot(t_new, x_interp); hold on;
plot(t_ref, x_ref, 'k:');
title('After interpolation filter — smooth'); xlim([0 0.06]);

subplot(4,1,4);
plot(t_new(1:length(x_resamp)), x_resamp); hold on;
plot(t_ref, x_ref, 'k:');
title('resample() polyphase — most accurate'); xlim([0 0.06]);

%% Spectrum: show images
N = length(x_stuffed);
f_ax = (0:N/2-1)*fs_new/N;
figure('Name','Zero Stuffing Spectrum');
plot(f_ax, abs(fft(x_stuffed))*2/N, 'DisplayName','Zero stuffed');
hold on;
Na = length(x_interp);
plot((0:Na/2-1)*fs_new/Na, abs(fft(x_interp))*2/Na, 'r', 'DisplayName','After interp filter');
for k = 0:L; xline(fs/2 + k*fs, 'k:', sprintf('%dHz', round(fs/2+k*fs))); end
legend; xlabel('Hz'); ylabel('|X|');
title('Spectral images from zero stuffing → removed by interpolation filter');
xlim([0 fs_new/2]);
💡 Hint

The spectrum of the zero-stuffed signal shows the original 80 Hz peak plus images at 420 Hz, 580 Hz, 920 Hz... (at fs-80=420, fs+80=580, 2fs-80=920 etc.). The interpolation filter removes everything above fs/2=250 Hz, leaving only the 80 Hz baseband component.

Key Takeaways

  • Zero stuffing inserts $L-1$ zeros between each sample, increasing the sample rate to $L \cdot f_s$.
  • Results in spectral images — copies of the original spectrum at multiples of the original Nyquist.
  • Zero stuffing ≠ zero padding: stuffing goes between samples (time domain); padding appends to the end (for FFT).
  • After zero stuffing, always apply interpolation lowpass filter (cutoff = $\pi/L$, gain = $L$).
  • In MATLAB: upsample(x, L) for zero stuffing; resample(x, L, 1) for complete upsampling.
