7.3 — MATLAB Function Reference

Module 7 · Reference & Summary · ~20 min

Introduction

A categorized cheat-sheet of every MATLAB function used in the course, plus a few close relatives you'll reach for constantly. Each entry gives the calling syntax, what it does, and which topic uses it. Functions that need a toolbox are flagged.

Toolbox flags: [SPT] = Signal Processing Toolbox · [WT] = Wavelet Toolbox · everything else is base MATLAB. Check what you have with ver or license('test','Signal_Toolbox').

2 · Fourier / Spectral Analysis

FunctionWhat it doesTopic
fft(x) / fft(x,M)DFT of x; the optional M zero-pads (or truncates) to length M.2.3, 3.2
ifft(X)Inverse FFT — back to the time domain.2.3
abs(X) / angle(X)Magnitude and phase of complex spectrum.2.1
fftshift(X)Shift zero frequency to the centre for a two-sided plot.2.1
unwrap(phase)Remove $2\pi$ jumps so phase plots are continuous.5.1
pwelch(x,...) [SPT]Welch power spectral density — averaged, lower-variance spectrum estimate.3.1
periodogram(x,...) [SPT]Single-record PSD estimate.3.1
Canonical one-sided amplitude spectrum (memorize this block): 
N  = length(x);
X  = fft(x);
P2 = abs(X)/N;            % two-sided, normalized
P1 = P2(1:floor(N/2)+1);  % one-sided
P1(2:end-1) = 2*P1(2:end-1);   % double interior bins
f  = (0:floor(N/2)) * fs/N;    % frequency axis (Hz)
plot(f, P1);

7 · Resampling & Interpolation

FunctionWhat it doesTopic
upsample(x,L) [SPT]Zero-stuff: insert L−1 zeros between samples (no filtering).6.2, 6.4
downsample(x,M) [SPT]Keep every M-th sample (no anti-alias filter — beware!).6.3
interp(x,L) [SPT]Upsample + built-in lowpass = proper interpolation by L.6.2
decimate(x,M) [SPT]Lowpass + downsample = proper decimation by M.6.3
resample(x,L,M) [SPT]Rational rate change by L/M with anti-alias filtering (one-stop shop).6.1
interp1(x,v,xq,method)General 1-D interpolation ('linear','spline','pchip'). Base MATLAB.3.4
Mapping operations to the right call: "make the rate higher correctly" → interp/resample, not bare upsample. "make it lower correctly" → decimate/resample, not bare downsample. The bare functions only do the zero-stuffing/sample-dropping step and leave you to add the filter.

Quick Self-Check

Q1. You need zero phase distortion on an already-recorded signal. Use:

  • filter(b,a,x)
  • filtfilt(b,a,x)
  • conv(x,b)
  • fft(x)

Q2. To safely reduce the sample rate by 4 without aliasing, prefer:

  • downsample(x,4)
  • decimate(x,4) (or resample)
  • upsample(x,4)
  • fft(x,4)

Q3. Which function visualizes poles, zeros, and the unit circle?

  • freqz
  • spectrogram
  • zplane
  • pwelch

Key Takeaways

  • Spectra: fft → abs/angle → one-sided scaling; estimate power with pwelch.
  • Design FIR with fir1/firpm, IIR with butter/cheby1/ellip; analyse with freqz/zplane/grpdelay.
  • Apply with filter (real-time) or filtfilt (zero-phase, offline).
  • Resample correctly with interp/decimate/resample — the bare upsample/downsample skip the filter.
