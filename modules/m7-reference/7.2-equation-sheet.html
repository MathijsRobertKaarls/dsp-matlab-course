7.2 — Master Equation Sheet

Module 7 · Reference & Summary · ~20 min

Introduction

Every important equation in the course, in one place, with each symbol named and a one-line note on when to use it. This is your exam crib sheet and your day-to-day lookup. Equations are grouped to follow the modules.

Symbol conventions used throughout: $x[n]$ = input samples, $y[n]$ = output samples, $h[n]$ = impulse response, $X(\cdot)$ = transform of $x$, $f_s$ = sample rate (Hz), $T_s = 1/f_s$ = sample period, $N$ = number of samples, $\omega = 2\pi f/f_s$ = normalized angular frequency (rad/sample).

1 · Sampling

Sample period & rate:

$$T_s = \frac{1}{f_s}, \qquad t_n = n T_s = \frac{n}{f_s}$$

Nyquist criterion — sample faster than twice the highest frequency:

$$f_s > 2 f_{max} \qquad\Longleftrightarrow\qquad f_{max} < \frac{f_s}{2}$$

Aliased frequency — what an undersampled tone at $f$ appears as:

$$f_{alias} = \left| f - f_s \cdot \mathrm{round}\!\left(\frac{f}{f_s}\right)\right| \le \frac{f_s}{2}$$

$f$ = true frequency, $f_s$ = sample rate. If $f < f_s/2$ there is no aliasing. (Topics 1.2, 2.2)

2 · LTI Systems & Convolution

Convolution — output of any LTI system:

$$y[n] = (x * h)[n] = \sum_{k=-\infty}^{\infty} x[k]\, h[n-k]$$

Linearity (superposition) and time-invariance:

$$\mathcal{T}\{a\,x_1 + b\,x_2\} = a\,\mathcal{T}\{x_1\} + b\,\mathcal{T}\{x_2\}, \qquad x[n-n_0] \;\to\; y[n-n_0]$$

$h[n]$ = impulse response; convolution in time = multiplication in frequency. (Topic 1.4)

3 · Fourier Transforms

Continuous Fourier Transform (CTFT):

$$X(f) = \int_{-\infty}^{\infty} x(t)\, e^{-j 2\pi f t}\, dt$$

Discrete-Time Fourier Transform (DTFT):

$$X(e^{j\omega}) = \sum_{n=-\infty}^{\infty} x[n]\, e^{-j\omega n}$$

Discrete Fourier Transform (DFT) — forward and inverse:

$$X[k] = \sum_{n=0}^{N-1} x[n]\, e^{-j 2\pi k n / N}, \qquad x[n] = \frac{1}{N}\sum_{k=0}^{N-1} X[k]\, e^{+j 2\pi k n / N}$$

Bin-to-frequency mapping:

$$f_k = \frac{k\, f_s}{N}, \qquad k = 0, 1, \dots, N-1$$

Frequency resolution (true, set by record length):

$$\Delta f = \frac{f_s}{N} = \frac{1}{N T_s} = \frac{1}{T_{record}}$$

$X[k]$ = DFT coefficient (bin $k$), $N$ = number of samples, $T_{record} = N T_s$ = total observation time. The FFT computes $X[k]$ in $O(N\log N)$. (Topics 2.1, 2.3, 3.2)

Real-signal symmetry: for real $x[n]$, $X[N-k] = X[k]^{*}$. Only bins $0 \ldots N/2$ are unique — that's why one-sided spectra multiply amplitude by 2 (except DC and Nyquist).

4 · Signal Quality

Power & amplitude in decibels:

$$\text{dB}_{power} = 10\log_{10}\!\frac{P}{P_{ref}}, \qquad \text{dB}_{amp} = 20\log_{10}\!\frac{A}{A_{ref}}$$

Signal-to-Noise Ratio:

$$\text{SNR}_{dB} = 10\log_{10}\!\frac{P_{signal}}{P_{noise}} = 20\log_{10}\!\frac{A_{signal,\,rms}}{A_{noise,\,rms}}$$

SNR gain from averaging $M$ records of a repeatable signal:

$$\text{SNR}_{improvement} = \sqrt{M} \quad\Longleftrightarrow\quad 10\log_{10} M \;\text{dB} \;\;(\approx 3\text{ dB per doubling})$$

$P$ = power, $A$ = amplitude, $M$ = number of averaged records. Coherent averaging cancels random noise but keeps the repeatable signal. (Topic 3.1)

Windowing — taper before the FFT to reduce leakage:

$$x_w[n] = w[n]\, x[n]$$

Common windows (length $N$, $0 \le n \le N-1$):

$$w_{Hann}[n] = 0.5\left(1 - \cos\frac{2\pi n}{N-1}\right), \qquad w_{Hamming}[n] = 0.54 - 0.46\cos\frac{2\pi n}{N-1}$$

$w[n]$ = window. Trade-off: wider main lobe (worse resolution) ↔ lower side lobes (less leakage). (Topic 3.3)

5 · Z-Transform & Filters

Z-transform:

$$X(z) = \sum_{n=-\infty}^{\infty} x[n]\, z^{-n}, \qquad z = r e^{j\omega}$$

Frequency response from the transfer function (evaluate on the unit circle $z = e^{j\omega}$):

$$H(e^{j\omega}) = H(z)\big|_{z = e^{j\omega}}$$

Delay property (the workhorse of difference equations):

$$x[n-k] \;\xrightarrow{\;\mathcal{Z}\;}\; z^{-k} X(z)$$

General difference equation (ARMA):

$$y[n] = \sum_{k=0}^{M} b_k\, x[n-k] \;-\; \sum_{k=1}^{N} a_k\, y[n-k]$$

FIR special case (no feedback, $a_k = 0$ for $k\ge1$):

$$y[n] = \sum_{k=0}^{M} b_k\, x[n-k]$$

Transfer function (take the Z-transform of the difference equation):

$$H(z) = \frac{Y(z)}{X(z)} = \frac{\displaystyle\sum_{k=0}^{M} b_k\, z^{-k}}{\displaystyle 1 + \sum_{k=1}^{N} a_k\, z^{-k}} = \frac{b_0\displaystyle\prod_{i}(1 - q_i z^{-1})}{\displaystyle\prod_{i}(1 - p_i z^{-1})}$$

$b_k$ = feedforward (numerator) coefficients, $a_k$ = feedback (denominator) coefficients, $q_i$ = zeros, $p_i$ = poles, $M$/$N$ = numerator/denominator orders. (Topics 5.1–5.3, 5.6)

Stability rule: a causal LTI filter is stable $\iff$ all poles satisfy $|p_i| < 1$ (strictly inside the unit circle). Zeros may lie anywhere. FIR filters have no poles (except at the origin) and are therefore always stable.

Magnitude (Bode) and phase:

$$|H(e^{j\omega})|_{dB} = 20\log_{10}\big|H(e^{j\omega})\big|, \qquad \phi(\omega) = \angle H(e^{j\omega})$$

Group delay — the per-frequency time delay:

$$\tau_g(\omega) = -\frac{d\,\phi(\omega)}{d\omega}$$

Linear phase $\Rightarrow$ $\phi(\omega) = -\alpha\omega \Rightarrow$ constant group delay $\tau_g = \alpha$ (a symmetric FIR of order $L$ gives $\tau_g = L/2$ samples). (Topics 5.4, 5.5)

6 · Resampling

Zero stuffing (upsample by $L$):

$$x_L[n] = \begin{cases} x[n/L], & n = 0, L, 2L, \dots \\ 0, & \text{otherwise} \end{cases} \qquad\xrightarrow{\;\mathcal{Z}\;}\; X_L(z) = X(z^L)$$

Downsampling (decimate by $M$): lowpass to $f_s/(2M)$ first, then keep every $M$-th sample:

$$y[n] = x_{filtered}[nM]$$

New sample rates:

$$f_{s,\,up} = L\, f_s, \qquad f_{s,\,down} = \frac{f_s}{M}, \qquad f_{s,\,rational} = \frac{L}{M}\, f_s$$

Interpolation filter gain after zero-stuffing by $L$ (restore amplitude):

$$h_{interp}[n] = L \cdot h_{LPF}[n], \quad \text{cutoff } = \frac{f_s}{2}\ \text{(i.e. } \tfrac{\pi}{L}\ \text{rad/sample)}$$

$L$ = upsampling factor, $M$ = downsampling factor. Zero-stuffing creates $L-1$ spectral images per band; the $\times L$ lowpass removes them and corrects the amplitude. (Topics 6.2–6.4)

  • $k = 150$

  • $k = 75$

  • 3 dB
  • 20 dB
  • 10 dB

Q3. In the z-domain, upsampling (zero-stuffing) by $L$ transforms $X(z)$ into:

  • $X(z)/L$
  • $X(z - L)$
  • $X(z^L)$
  • $L\,X(z)$

Key Takeaways

  • Sampling lives or dies on $f_s > 2f_{max}$; the FFT axis is $f_k = k f_s/N$ with true resolution $\Delta f = f_s/N$.
  • Filters are summarized by $H(z) = \frac{\sum b_k z^{-k}}{1+\sum a_k z^{-k}}$; stability needs all poles inside the unit circle.
  • Averaging buys $\sqrt{M}$ SNR; zero-stuffing by $L$ does $X(z)\to X(z^L)$ and needs a $\times L$ lowpass.
