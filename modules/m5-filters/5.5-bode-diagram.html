5.5 — Bode Diagram

Module 5 · Filter Design · ~20 min

Introduction

The Bode diagram is the standard way to visualize a filter's frequency response — plotting magnitude and phase separately over a logarithmic frequency axis. Logarithmic scaling is crucial: it lets you see decades of frequency range at once and reveals straight-line asymptotic behavior that makes filter design intuitive.

Core Concept

Two Plots in One

A Bode diagram consists of two plots sharing the same (logarithmic) frequency axis:

  1. Magnitude plot: $20\log_{10}|H(e^{j\omega})|$ in dB vs. frequency (log scale)
  2. Phase plot: $\angle H(e^{j\omega})$ in degrees vs. frequency (log scale)

Key Features to Read from a Bode Plot

  • Passband gain: the flat region (ideally 0 dB for unit gain)
  • Cutoff frequency $f_c$: conventionally where the gain drops to $-3$ dB (half power)
  • Rolloff rate: how steeply the magnitude drops in the stopband, in dB/decade or dB/octave. Each first-order pole contributes −20 dB/decade. A 4th-order Butterworth rolls off at −80 dB/decade.
  • Phase at cutoff: for a Butterworth, exactly −45° × order at $f_c$

Rolloff Rates

  • 1st-order system: −20 dB/decade = −6 dB/octave
  • 2nd-order system: −40 dB/decade = −12 dB/octave
  • $N$th-order: $-20N$ dB/decade

Mathematical Foundation

For a first-order continuous-time lowpass: $H(s) = \frac{1}{1 + s/\omega_c}$

Magnitude: $|H(j\omega)| = \frac{1}{\sqrt{1 + (\omega/\omega_c)^2}}$

In dB: $20\log_{10}|H| \approx \begin{cases} 0 \text{ dB} & \omega \ll \omega_c \\ -20\log_{10}(\omega/\omega_c) & \omega \gg \omega_c \end{cases}$

For discrete-time filters, replace $s \to (z-1)/T$ (bilinear transform) or evaluate $H(e^{j\omega})$ directly.

Conceptual Questions

Q1. At the −3 dB cutoff frequency, the filter's amplitude gain is approximately:

  • 0 (complete rejection)
  • 0.5 (half the passband amplitude)
  • 0.707 (= 1/√2, half the passband power)
  • 1.0 (unchanged)

Q2. An 8th-order Butterworth lowpass filter has a stopband rolloff of:

  • −20 dB/decade
  • −80 dB/decade
  • −160 dB/decade
  • −8 dB/decade

Q3. Why is the frequency axis of a Bode plot logarithmic?

  • To make the magnitude response appear linear
  • To display a wide frequency range and reveal the straight-line asymptotic rolloff behavior
  • Because decibels require a log scale
  • To match the human perception of pitch

MATLAB Exercise

%% 5.5 Bode Diagrams of Digital Filters

fs = 10000;   % Higher fs for wide frequency range

%% Part A — Compare Butterworth orders
figure('Name','Butterworth Order Effect');
orders = [1, 2, 4, 8];
colors = {'b','r','g','m'};
Fc = 1000;  Fn = Fc/(fs/2);

subplot(2,1,1); hold on;
subplot(2,1,2); hold on;

for i = 1:4
  [b, a] = butter(orders(i), Fn, 'low');
  [H, f] = freqz(b, a, 4096, fs);
  mag_db = 20*log10(abs(H));
  phase_deg = unwrap(angle(H))*180/pi;

  subplot(2,1,1);
  semilogx(f, mag_db, colors{i}, 'DisplayName', sprintf('Order %d (%.0fdB/dec)', orders(i), 20*orders(i)));

  subplot(2,1,2);
  semilogx(f, phase_deg, colors{i}, 'DisplayName', sprintf('Order %d', orders(i)));
end

subplot(2,1,1);
xline(Fc,'k--','fc'); yline(-3,'k:','-3dB');
xlabel('Frequency (Hz)'); ylabel('Magnitude (dB)');
title('Butterworth Lowpass — Effect of Filter Order'); legend; ylim([-100 5]); grid on;

subplot(2,1,2);
xline(Fc,'k--','fc');
xlabel('Frequency (Hz)'); ylabel('Phase (degrees)');
title('Phase Response'); legend; grid on;

%% Part B — Compare filter types (same order, same cutoff)
figure('Name','Filter Type Comparison');
[b_butt,  a_butt]  = butter(4,  Fn, 'low');
[b_cheb1, a_cheb1] = cheby1(4, 1, Fn, 'low');   % 1 dB passband ripple
[b_cheb2, a_cheb2] = cheby2(4, 40, Fn, 'low');   % 40 dB stopband attenuation
[b_ellip, a_ellip] = ellip(4, 1, 40, Fn, 'low'); % 1dB pass, 40dB stop ripple

filters = {b_butt,a_butt; b_cheb1,a_cheb1; b_cheb2,a_cheb2; b_ellip,a_ellip};
names   = {'Butterworth','Chebyshev I (1dB)','Chebyshev II (40dB)','Elliptic'};

for i = 1:4
  [H, f] = freqz(filters{i,1}, filters{i,2}, 4096, fs);
  semilogx(f, 20*log10(abs(H)), 'DisplayName', names{i}); hold on;
end
xline(Fc,'k--','fc'); yline(-3,'k:'); yline(-40,'k:','-40dB');
xlabel('Hz'); ylabel('dB'); title('4th-Order Lowpass: Four Classic Types');
legend; ylim([-80 5]); grid on;

%% Part C — Read the Bode plot numerically
[b4, a4] = butter(4, Fn, 'low');
[H4, f4] = freqz(b4, a4, 4096, fs);
mag4 = 20*log10(abs(H4));

% Find -3dB frequency
[~, idx] = min(abs(mag4 - (-3)));
fprintf('\n4th-order Butterworth -3dB frequency: %.1f Hz\n', f4(idx));

% Find -20dB frequency
[~, idx20] = min(abs(mag4 - (-20)));
fprintf('-20dB frequency: %.1f Hz\n', f4(idx20));
fprintf('Rolloff per decade from fc to -20dB: %.0f dB/decade\n', ...
  (mag4(idx20)-mag4(idx)) / log10(f4(idx20)/f4(idx)));
💡 Hint

Part B shows the classic filter type trade-offs. Butterworth: maximally flat passband, no ripple. Chebyshev I: ripple in passband, steeper rolloff. Chebyshev II: ripple in stopband, flat passband. Elliptic: ripple in both, steepest possible rolloff for given order — most efficient. The measured rolloff in Part C should be close to the theoretical 80 dB/decade for a 4th-order filter.

Key Takeaways

  • Bode diagram: magnitude (dB) and phase (degrees) vs. frequency on a log scale.
  • Cutoff frequency $f_c$: where magnitude = −3 dB (half-power point).
  • Rolloff rate: $-20N$ dB/decade for an $N$th-order filter.
  • Filter types: Butterworth (flat), Chebyshev (steeper), Elliptic (steepest) — different ripple trade-offs.
  • In MATLAB: freqz(b,a,N,fs) then plot with semilogx; or use bode() for s-domain.
