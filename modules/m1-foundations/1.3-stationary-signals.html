1.3 — Stationary & Periodic Signals

Module 1 · Foundations · ~20 min

Introduction

Before we can process signals, we need to describe them precisely. Two fundamental properties of signals that shape how we analyze them are periodicity (does it repeat?) and stationarity (does its character stay the same over time?).

A musical note held steady is both periodic and stationary. A spoken sentence is neither — its frequency content changes with every phoneme. Choosing the right analysis tool depends on understanding these properties.

Core Concept

Periodic Signals

A signal $x(t)$ is periodic with period $T_0$ if:

$$x(t + T_0) = x(t) \quad \text{for all } t$$

Key properties of a periodic signal:

  • Period $T_0$ (seconds): time for one full cycle
  • Fundamental frequency $f_0 = 1/T_0$ (Hz): how many cycles per second
  • Angular frequency $\omega_0 = 2\pi f_0$ (radians/second)

The Sine Wave — Building Block of DSP

The simplest periodic signal is a sinusoid:

$$x(t) = A \sin(2\pi f_0 t + \phi)$$

where:

  • $A$ — Amplitude: peak value (units of the signal, e.g. Volts)
  • $f_0$ — Frequency (Hz): how fast it oscillates
  • $\phi$ — Phase (radians): time shift relative to a reference

Phase — A Common Source of Confusion

Phase $\phi$ shifts the signal in time. A phase of $\pi/2$ radians shifts the sine into a cosine:

$$\sin\!\left(2\pi f_0 t + \frac{\pi}{2}\right) = \cos(2\pi f_0 t)$$

Phase becomes critical when combining signals — two identical sine waves that are exactly 180° out of phase ($\phi = \pi$) will cancel each other completely. This is the principle behind noise-cancelling headphones!

Stationary vs. Non-Stationary Signals

A signal is stationary if its statistical properties (mean, variance, frequency content) do not change over time.

  • Stationary: a pure tone held at constant volume, white noise, a DC offset
  • Non-stationary: speech, music, an ECG during exercise, a chirp (sweeping frequency)
Why it matters: The standard Fourier Transform assumes the signal is stationary over all time. For non-stationary signals, we need the Short-Term Fourier Transform (Module 4) or wavelets — tools that reveal how frequency content changes over time.

Time-Domain Signal Parameters

When looking at a signal over time, we characterize it by:

  • Mean (DC component): the average value $\mu = \frac{1}{N}\sum_{n=0}^{N-1} x[n]$
  • Variance / Power: average squared deviation from mean
  • RMS (Root Mean Square): $x_{rms} = \sqrt{\frac{1}{N}\sum x[n]^2}$ — related to signal power

Mathematical Foundation

Discrete-time sinusoid

In MATLAB, signals are discrete. The discrete-time sinusoid is:

$$x[n] = A \sin(2\pi f_0 n T_s + \phi) = A \sin\!\left(\frac{2\pi f_0}{f_s} n + \phi\right)$$

The ratio $\hat{\omega} = 2\pi f_0 / f_s$ is the digital frequency in radians/sample. It ranges from $0$ to $\pi$ (representing 0 to $f_s/2$).

Superposition — Combining Sinusoids

Any periodic signal can be expressed as a sum of sinusoids (Fourier Series). For a square wave at frequency $f_0$:

$$x(t) = \frac{4}{\pi} \sum_{k=0}^{\infty} \frac{1}{2k+1} \sin\!\left(2\pi (2k+1) f_0 t\right)$$

= fundamental + 3rd harmonic/3 + 5th harmonic/5 + ...

MATLAB Exercise

Explore amplitude, frequency, and phase. Then build a signal from harmonics (Fourier synthesis).

%% 1.3 Signal Properties & Fourier Synthesis

fs = 8000;
t  = 0:1/fs:1-1/fs;   % 1 second

%% Part A — Effect of amplitude, frequency, phase
f0 = 200;   % Base frequency

x1 = 1.0 * sin(2*pi*f0*t);           % Reference
x2 = 0.5 * sin(2*pi*f0*t);           % Half amplitude
x3 = 1.0 * sin(2*pi*2*f0*t);         % Double frequency
x4 = 1.0 * sin(2*pi*f0*t + pi/2);    % 90° phase shift (cosine)

figure('Name','Sine Wave Parameters');
win = 1:round(fs/f0*3);  % Show 3 cycles
subplot(4,1,1); plot(t(win), x1(win)); title('Original: A=1, f=200Hz, φ=0'); ylim([-1.5 1.5]);
subplot(4,1,2); plot(t(win), x2(win)); title('Half amplitude: A=0.5');        ylim([-1.5 1.5]);
subplot(4,1,3); plot(t(win), x3(win)); title('Double frequency: f=400Hz');    ylim([-1.5 1.5]);
subplot(4,1,4); plot(t(win), x4(win)); title('90° phase shift (cosine)');     ylim([-1.5 1.5]);

%% Part B — Build a square wave from harmonics
f0   = 100;   % Fundamental frequency
N_harmonics = 15;  % Number of odd harmonics to include
x_sq = zeros(1, length(t));

figure('Name','Fourier Synthesis of Square Wave');
for k = 1:N_harmonics
  n = 2*k - 1;   % Odd harmonics: 1, 3, 5, ...
  x_sq = x_sq + (4/pi) * (1/n) * sin(2*pi*n*f0*t);
  if k <= 4 || k == N_harmonics
    subplot(1,1,1); cla;
    win2 = 1:round(fs/f0*3);
    plot(t(win2), x_sq(win2));
    title(sprintf('Square wave: %d harmonic(s) — up to %d Hz', k, n*f0));
    ylim([-1.5 1.5]); xlabel('Time (s)'); ylabel('Amplitude');
    drawnow; pause(0.4);
  end
end
💡 Hint

In Part B, watch the synthesis animation. With just 1 harmonic you see a sine. As you add more odd harmonics, the signal progressively looks more like a square wave. Notice that with 15 harmonics there's still some "ringing" near the edges — this is the Gibbs phenomenon, a fundamental limitation of truncated Fourier series.

Key Takeaways

  • A sinusoid is described by three parameters: amplitude $A$, frequency $f_0$, and phase $\phi$.
  • Periodic signals repeat with period $T_0 = 1/f_0$; any periodic signal is a sum of sinusoids (Fourier Series).
  • Stationary signals have time-invariant statistics; non-stationary signals (speech, music) require special analysis tools.
  • Phase shift of 180° ($\pi$) causes complete cancellation — the basis of noise cancellation.
  • The RMS of a unit-amplitude sine wave is $1/\sqrt{2} \approx 0.707$.
